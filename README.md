> **B2B External Attack Surface Management (EASM) SaaS** — AI-powered threat intelligence, real paying subscribers, fully deployed in production. This is my MCA capstone and my primary product.

```
┌──────────────────────────────────────────────────────────────┐
│  Next.js 15 Frontend (TypeScript)                            │
│  ├─ Marketing: animated canvas globe, live telemetry cards   │
│  ├─ Dashboard: Network Shield · Threat Intel · Security Vault│
│  └─ Reports: Recharts visualizations wired to real Supabase  │
├──────────────────────────────────────────────────────────────┤
│  Express Backend (TypeScript)                                │
│  ├─ 18-intent Gemini 2.0 Flash AI terminal                   │
│  ├─ IDOR-hardened REST APIs across every endpoint             │
│  ├─ Webhook-verified Dodo Payments subscription flow          │
│  └─ Rate limiter — trust-proxy bug fixed after k6 @ 1000 VUs │
├──────────────────────────────────────────────────────────────┤
│  Infra: Supabase Postgres · Clerk (OTP + Google OAuth)       │
│         Vercel (frontend) · Render (backend)                 │
└──────────────────────────────────────────────────────────────┘
```

**Engineering wins:**
- 🔍 Found & patched a rate-limiter trust-proxy misconfiguration via a k6 load test at **1,000 virtual users**
- 🔒 Closed IDOR gaps across every API endpoint (object-level authorization)
- 🎨 Ripped out a third-party skeleton library and shipped a custom shimmer system for perf
- 📊 Killed all hardcoded dashboard data — everything reads from live Supabase now
